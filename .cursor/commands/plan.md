# Plan Command
**Goal:** Generate a detailed implementation plan for the next feature.

**Instructions:**
1. Read the first 5 lines of @TASKS.md to find the "Current Focus Ticket".
2. Read the specific section for that ticket in @TASKS.md (Do NOT read the whole file).
3. Update the ticket section to set `Status: In Progress` before generating the plan.
4. Create/Update @current-plan.md using the template @.cursor/commands/plan-template.md. ensure alignment to @PRD.md.
5. **Crucial:** The plan must be detailed (file paths, function names, test steps).
6. Ask the user for approval before proceeding to execution.
